# Beautiful Comments

Here, It has list of beautiful and simple comments for specific programming language.

It gives an overview of the comment styles, Adjust or change comments syntax according to programming language used.

For descriptive purpose, It has denoted in JavaScript multi-line comments.

  ______________________________________________________________________________________________
